# Sakshi Bagora - Portfolio Website

## Project Overview

- **Project Type**: React + TypeScript Portfolio Website
- **Entry Point**: `src/main.tsx` (React application entry)
- **Build System**: Vite 7.0.0 (Fast development and build)
- **Styling System**: Tailwind CSS 3.4.17 (Atomic CSS framework)
- **Animation**: Framer Motion 11.0.8

## Features Implemented

### Sections
1. **Hero Section** - Animated introduction with parallax effects, floating shapes, and professional branding
2. **About Section** - Personal journey, contact information, education, and certifications
3. **Skills & Expertise** - Grid layout showcasing core competencies and platform expertise
4. **Projects Section** - Featured projects with impact metrics and tags
5. **Clients Marquee** - Animated scrolling logos of brands worked with
6. **Blog Section** - Latest insights and articles
7. **Contact Section** - Contact form with validation and social links
8. **Footer** - Quick links and social connectivity

### Key Features
- Smooth scrolling navigation
- Scroll-triggered animations (fade-in, slide-up)
- Responsive design (mobile, tablet, desktop)
- Animated client logo marquee
- Interactive hover effects
- Custom scrollbar styling
- Form validation with loading states

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Production Build
```bash
npm run build
```

## Customization

To customize this portfolio:
1. Update the client logos in the `clients` array in `src/App.tsx`
2. Modify the `skills`, `projects`, and `blogPosts` arrays
3. Update contact information in the Contact section
4. Replace the profile placeholder in the Hero section
